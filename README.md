## 💡 Советы по прохождению роадмапа

- **Не решайте задачи подряд** — группируйте их по паттернам, чтобы видеть структуру.
- **Ведите дневник ошибок**: фиксируйте, где ошиблись, и возвращайтесь к этим задачам через пару дней.
- **Проговаривайте решения вслух** — это важно для интервью.
- **Используйте только стандартный редактор LeetCode** — без автодополнений и подсказок.
- **В день интервью не решайте новые задачи** — только повторяйте и отдыхайте.

## ⚡ Что важно помнить

- На собеседовании в Яндекс часто встречаются задачи на *Two Pointers*, *Sliding Window*, *BFS/DFS*, *Backtracking* и *Binary Search*.
- Важно не только решить задачу, но и уметь объяснить выбор структуры данных и оценить сложность по времени и памяти.
- Не заучивайте решения — учите паттерны и подходы.
